复制一个单词 光标移到想要被复制词的词首，输入yw 单词任意位置 yaw

删除一个单词 光标移到想要被复制词的词首，输入dw 单词任意位置 daw

分屏

命令行分屏 vim -o2 file1 file2 水平分屏 vim -O2 file1 file2 垂直分屏

编辑中分屏 ctrl+w v左右分割当前文件 ctrl+w s上下分割当前文件 :sp filename 上下分割，并打开新的文件 :vsp filename 左右分割，并打开一个新的文件

分屏编辑中光标的移动 ctrl+w kjlhw

移动分屏 ctrl+w KJLH

基本配置

set nocompatible filetype plugin on syntax on "语法高亮 set number "显示行号 set showmode "在底部显示处于命令模式还是插入模式 set showcmd "命令模式下，在底部显示当前键入的指令 set mouse=a "支持使用鼠标 set encoding=utf-8 set autoindent "按下回车键后，下一行的缩进会自动和上一行保持一致 set tabstop=4 "tab键显示的空格数 set shiftwidth=4 "按>>或<<或==调整缩进时,每一级的字符数 set expandtab "自动将tab转为空格

set cursorline "光标所在当前行高亮
